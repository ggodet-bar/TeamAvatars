Décrit l'ensemble des classes constituant la partie abstraite de l'Objet Interactionnel Entité BoitePropriete.

Description de la BoitePropriete

Une BoitePropriete correspond à une boîte flottante, constituée d'un premier panel décrivant un titre ainsi qu'un titre étendu, ainsi que d'un ensemble de panels généralistes de deux types :

  • Des panels contenant des lignes de texte. Chaque ligne peut être colorée indépendamment. Le formattage de ces lignes peut être réalisé selon les conventions HTML ou ANSI standard. Si le nombre de lignes dépasse la taille du panel, un ascenseur vertical apparaît,
  • Des panels contenant des images. De même que pour le panel texte, un ascenseur apparaît si le nombre d'images contenu dans le panel dépasse la taille de ce dernier. A IMPLEMENTER PLUS PRECISEMENT !!
Il est à noter que chaque panel est constitué d'un titre et de son contenu.

Une BoitePropriete peut être présentée selon deux modes :

  • BoitePropriete réduite : Seul le titre de la boîte apparaît
  • BoitePropriete développée : Le titre et le titre étendu apparaissent, ainsi que tous les panels expressément indiqué comme étant actifs
Lorsque la BoitePropriete est en mode développée, les opérations suivantes peuvent être appliquées aux panels :
  • Propriétés étendues : Le titre du panel ainsi que son contenu sont affichés
  • Propriétés réduites : Seul le titre du panel apparaît
En mode développée, quatre types d'opération peuvent être appliqués pour l'affichage des panels de la BoiePropriete :
  • Panel x affiché : Affichage du panel
  • Panel x caché : Dissimulation du panel
  • Afficher tous les panels
  • Cacher tous les panels

Structure du package

Les classes sont structurées de la manière suivante :

  • L'interface BoitePropriete décrit les services spécifiques à l'Objet BoitePropriete
  • La classe BoiteProprieteImpl décrit l'implémentation des services de l'BoitePropriete. Cette classe implémente également les services propres au type EntityObject, ainsi que les services liés à l'affichage de l'Objet Interactionnel Displayable
  • La classe BoiteProprieteFactory gère l'instanciation des objets BoitePropriete
  • L'interface BoiteProprieteFactoryInterface sert à la déclaration des services de fabrique dans le cadre de l'implémentation OSGi
  • La classe Panel correspond à une classe < Partie > de l'Objet BoitePropriete
  • La classe BlocImage correspond également à une classe < Partie > de l'Objet BoitePropriete, utilisée pour la description des panels contenant des images.
  • La classe BoiteProprieteTechInterface décrit l'ensemble des services pour l'affichage de la BoitePropriete. Cette interface doit être implémentée par tout service réalisant la représentation graphique de la BoitePropriete
